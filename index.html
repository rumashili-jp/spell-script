🌱 1. プログラミングってなーに？

プログラミングは、コンピュータへの「おねがいごと」を紙に書くようなものよ。
この「Spell-Script」では、あなたが書いた1行1行が「魔法の呪文」になって、コンピュータがそれを順番に実行してくれるの。

✨ 2. これだけは知っておきたい基本

💬 文字を表示する (log)

コンピュータに何かしゃべらせたい時は log("こんにちは") と書きましょう。

📦 データをしまう箱 (変数/setvar)

setvar("なまえ", "アリス") と書くと、「なまえ」という箱にデータをしまっておけるわよ。

⚡ 3. 魔法の記号（演算子）

数字を計算したり、大きさを比べたりするための記号よ。これらを組み合わせることで、複雑な魔法が作れるようになるわ！

算数の記号：

+ (たす)
- (ひく)
* (かける)
/ (わる)
% (あまり)

比べっこの記号（if文でよく使うわ！）：

== (おなじ)
!= (ちがう)
> (より大きい)
< (より小さい)
>= (以上)
<= (以下)
if(level >= 10) log("上級魔法が使えるよ！") ifend

🪄 4. 魔法の言葉（命令）の使いかた

💎 log(内容)

カッコの中に書いたものを「Log」タブに書き出します。文字をつなげる時は `+` を使ってね。

log("答えは" + (10 + 20) + "です")

💎 setvar("名前", 値)

「名前」という箱を作って、そこに「値」を保存します。

setvar("hp", 100) setvar("items", ["杖", "薬草"])

💎 if(条件) ... elif ... else ... ifend

「もし〜なら」を判定します。`elif`（そうでなくて、もし）や `else`（どれでもなければ）で細かく分けられるわ。最後は必ず `ifend` で締めてね。

if(hp == 0) log("たおれてしまった！") elif(hp < 20) log("ピンチ！回復して！") else log("まだまだ元気よ！") ifend

💎 while(条件) ... whileend

条件が正しい間、ずっと繰り返します。終わったら `whileend` と書きましょう。

setvar("n", 1) while(n <= 3) log(n + "回目の魔法！") setvar("n", n + 1) whileend

💎 func 名前(引数) ... funcend

あなただけの新しい魔法（関数）を作ります。何度も使う処理をまとめておくと便利よ！

func attack(power) log(power + "のダメージ！") funcend attack(50)

🌱 1. プログラミングってなーに？

プログラミングは、コンピュータへの「おねがいごと」を紙に書くようなものだよ。この「Spell-Script」では、あなたが書いた1行1行が「魔法の呪文」になって、コンピュータがそれを順番に実行してくれるんだ。間違えても爆発したりしないから、安心して色々な呪文を試してみてね！

✨ 2. これだけは知っておきたい基本

💬 文字を表示する (log)

コンピュータに何かしゃべらせたい時は log("こんにちは") と書こう。文字は必ず " (ダブルクォーテーション)で囲むのがルールだよ。

📦 データをしまう箱 (変数/setvar)

setvar("なまえ", "アリス") と書くと、「なまえ」という名前の箱にデータをしまっておけるよ。後で log(なまえ) と書けば、中身を取り出せるよ。

💡 魔法使いのアドバイス：
箱の名前（変数名）は必ず「文字」から始めてね。数字から始めると、コンピュータが「箱の名前」なのか「ただの数字」なのか勘違いしてパニックになっちゃうんだ！

⚡ 3. 魔法の記号（演算子）

数字を計算したり、状態を比べたりするための大切な記号たちだよ。

  • + - * / ： 足し算、引き算、掛け算、割り算。
  • == ： 左右が同じかどうか。
  • != ： 左右が違うかどうか。
  • > < ： どちらが大きいか比べる。
  • >= <= ： 以上・以下。
setvar("total", 10 + 20) log(total) // 30 と表示されるよ if(total >= 30) log("合格！") ifend

🪄 4. 魔法の言葉（命令）の使いかた

呪文の流れをコントロールするための特別な命令だよ。必ず最後は「エンド」で閉じるのを忘れないでね。

📚 複数のデータをまとめる (配列)

たくさんのデータを1つの変数にまとめてしまいたい時に使うよ。中身を取り出す時は、0番目から数えるのが魔法使いの常識だよ！

setvar("items", ["薬草", "聖水", "毒消し"]) log(items[0]) // 薬草 と表示されるよ log(items[1]) // 聖水 と表示されるよ

分岐の魔法 (if / ifend)

条件が合っている時だけ中身を実行するよ。

if(hp < 20) log("ピンチ！") ifend

繰り返しの儀式 (while / whileend)

条件が合っている間、ずっと中身を繰り返すよ。

setvar("n", 1) while(n <= 3) log(n + "回目！") setvar("n", n + 1) whileend

オリジナルの魔法 (func / funcend)

一連の動きを「新しい呪文」として登録できるよ。登録した後は、名前を呼ぶだけで実行できるんだ！

func fire() log("メラメラ...") log("Burn!!") funcend fire() // ここで呪文を呼び出す

💡 5. 上達のコツ

  • // コメントを活用して： // の後はメモ書きとして無視されるよ。自分用のメモを残しておくと読みやすいよ。
  • 1つずつ実行して： 一気にたくさん書かずに、少し書いたら「実行」して動きを確かめるのが一番の進道だよ。
  • エラーを恐れないで： 赤い文字が出たら、それはコンピュータからの「ヒント」だよ。カッコの閉じ忘れや、綴りの間違いがないか見てみてね。

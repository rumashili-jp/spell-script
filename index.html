🌱 1. プログラミングってなーに？

プログラミングは、コンピュータへの「おねがいごと」を紙に書くようなものよ。この「Spell-Script」では、あなたが書いた1行1行が「魔法の呪文」になって、コンピュータがそれを順番に実行してくれるの。間違えても爆発したりしないから、安心して色々な呪文を試してみてね！

✨ 2. これだけは知っておきたい基本

💬 文字を表示する (log)

コンピュータに何かしゃべらせたい時は log("こんにちは") と書きましょう。文字は必ず " (ダブルクォーテーション)で囲むのがルールよ。

📦 データをしまう箱 (変数/setvar)

setvar("なまえ", "アリス") と書くと、「なまえ」という名前の箱にデータをしまっておけるわ。後で log(なまえ) と書けば、中身を取り出せるの。

⚡ 3. 魔法の記号（演算子）

数字を計算したり、状態を比べたりするための大切な記号たちよ。

  • + - * / ： 足し算、引き算、掛け算、割り算。
  • == ： 左右が同じかどうか。
  • != ： 左右が違うかどうか。
  • > < ： どちらが大きいか比べる。
  • >= <= ： 以上・以下。
setvar("total", 10 + 20) log(total) // 30 と表示されるわ if(total >= 30) log("合格！") ifend

🪄 4. 魔法の言葉（命令）の使いかた

呪文の流れをコントロールするための特別な命令よ。必ず最後は「エンド」で閉じるのを忘れないでね。

分岐の魔法 (if / ifend)

条件が合っている時だけ中身を実行するわ。

if(hp < 20) log("ピンチ！") ifend

繰り返しの儀式 (while / whileend)

条件が合っている間、ずっと中身を繰り返すわ。

setvar("n", 1) while(n <= 3) log(n + "回目！") setvar("n", n + 1) whileend

オリジナルの魔法 (func / funcend)

一連の動きを「新しい呪文」として登録できるわ。登録した後は、名前を呼ぶだけで実行できるの！

func fire() log("メラメラ...") log("Burn!!") funcend fire() // ここで呪文を呼び出す

💡 5. 上達のコツ

  • // コメントを活用して： // の後はメモ書きとして無視されるわ。自分用のメモを残しておくと読みやすいわよ。
  • 1つずつ実行して： 一気にたくさん書かずに、少し書いたら「実行」して動きを確かめるのが一番の近道よ。
  • エラーを恐れないで： 赤い文字が出たら、それはコンピュータからの「ヒント」よ。カッコの閉じ忘れや、綴りの間違いがないか見てみてね。
